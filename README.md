## Pre-Requisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/) or [npm](https://npmjs.org)

## Set up development environment

- `yarn install` or `npm install` from root directory

## Local development

```
$ yarn dev
```

This will start a server on http://localhost:8080
